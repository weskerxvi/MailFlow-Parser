# Demo Assets

Place README images and GIFs in this folder.

Expected files:

- `swagger-docs.png`: screenshot of the deployed Swagger documentation at `/docs`.
- `processing-runs.png`: screenshot of `GET /processing-runs` response.
- `gmail-process.gif`: short GIF showing `POST /process/gmail` followed by `GET /orders`.
- `render-service.png`: optional screenshot of the Render service dashboard.
- `github-actions.png`: optional screenshot of GitHub Actions passing.

Keep images small enough for GitHub to load quickly.
